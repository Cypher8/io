图像处理

这是我较早的项目，利用python语言，对网站进行图像爬取后，实现了图像批量化的处理，包括图像的alpha混合、RGB分离蓝绿红通道、二像化、gamma校正、灰度化、图像旋转、svd奇异压缩等批量化处理

这种批量化的处理让我对图像处理有了更深刻的认知，这对以后的机器学习算法有了很好的基础

采销管理系统

这是一个综合企业管理系统，管理客户信息、销售合同、采购合同、技术打样、生产计划和库存等方面的数据。通过springMVC框架搭建，以及数据库的连接，实现数据的存储和增删改查功能

数据库实现的过程中，对于多种的数据组成的数据库，实例之间的关联关系比较重要，通过多个模块的子数据库的创建再合并在同一个数据库中，实现了数据的整合。这样更好的实现数据库的开发
